# Stellavia Construction Website

Premium animated React + Vite website for a construction company focused on flats and apartments.

## Tech Stack
- React
- Vite
- React Router DOM
- GSAP
- Framer Motion
- Lucide React

## Pages Included
- Home
- About
- Projects
- Floor Plans
- Contact

## Run Locally
```bash
npm install
npm run dev
```

## Build For Production
```bash
npm install
npm run build
```

## Deploy On Vercel
1. Upload this project to GitHub.
2. Import the repository in Vercel.
3. Framework preset: **Vite**
4. Build command: `npm run build`
5. Output directory: `dist`

`vercel.json` is included for React Router page refresh support.
